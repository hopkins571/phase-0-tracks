Semantic HTML tags are meant to clearly convey both the meaning of the tag to the browser and the developer. Rather than governing how content should be displayed (that's up to the stylesheet), these tags are used to show the meaning of individual sections within the greater context of the page. These tags can also be used by assistive software for differently abled users to identify the different elements in a page to help them access the content.
