Escaping Awkward Situations 101

Awkward situations happen every day. That's okay. By relying on a few simple strategies, you can effortlessly glide through the next awkward moment when it strikes. Interested? Here's a brief taste.

Step-by-step

  1. Identify the source of the awkwardness. Is it the other person? Is it you? Is it the other person's dog is doing?
  2. Deploy the appropriate exit stategy.
  3. Enjoy the sudden lack of awkwardness. And, if it didn't, remember that this man is far more awkward than you are.

Exit strategies

  • Use an invisibility cloak. Just pretend you're not there.
  • Spontaneously combust. Only to be used in emergencies!! Seriously, folks, this is the nuclear option.
